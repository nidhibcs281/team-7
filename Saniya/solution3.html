Pipeline Optimization in Residential Areas

SDG: 11

Target: 11.1

Indicator: 11.1.1

You are given a dataset that represents the cost of laying pipelines between houses in a residential area. Your task is to use Kruskal's Algorithm to find the Minimum Spanning Tree (MST), which will minimize the total cost of connecting all the houses using the pipeline.

The edges in the graph represent the cost to lay the pipeline between two houses.

Algorithm: Kruskal's Algorithm

Description: Finds the Minimum Spanning Tree (MST) by sorting all edges by weight and adding them to the tree while avoiding cycles.

Efficiency: O(|E| log|E|)
